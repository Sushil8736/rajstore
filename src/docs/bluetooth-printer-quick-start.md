## â“ FAQ

**Q: Do I need internet to print?**  
A: No, Bluetooth printing works offline.

**Q: Can I print to multiple printers?**  
A: Only one printer at a time. Disconnect current to switch.

**Q: Will it work on my phone?**  
A: Yes, on Android with Chrome. Not on iPhone/iOS.

**Q: Can I customize what prints?**  
A: The format is optimized for thermal receipts. Browser settings can be used for custom printing.

**Q: Do I need to install drivers?**
